## Descripción del Proyecto

Este proyecto presenta la **validación formal de un mecanismo de generación de alertas ante infracciones de veda pesquera**, con el objetivo de fortalecer la detección confiable de actividades ilegales y reducir falsos positivos o negativos en la vigilancia marítima.

El sistema integra evidencias provenientes de diversas fuentes (como sensores GPS, reportes y APIs externas), evalúa las condiciones mediante **reglas formales versionadas**, y genera alertas estructuradas con trazabilidad y justificación verificable.  

Dado que se trata de un sistema crítico, el modelo se desarrolló utilizando el lenguaje **VDM++** y se validó con la herramienta **VDM++ Toolbox**, aplicando técnicas de verificación formal, pre y post-condiciones, invariantes y pruebas reproducibles para garantizar la **correctitud y consistencia lógica** del mecanismo.

## Tecnologías y Herramientas
- Lenguaje formal: **VDM++**

- Paradigma: **Programación orientada a objetos**
- Método: **Validación formal y model checking**

---

## Objetivo General
Validar formalmente un mecanismo de generación de alertas ante infracciones de veda, asegurando detecciones precisas y notificaciones oportunas a las autoridades marítimas.

---

## Principales Componentes
- **Recepción y normalización de evidencias**  
- **Evaluación de reglas formales versionadas**  
- **Generación de alertas estructuradas con justificación**  
- **Notificación automática multicanal (email, API, SMS)**  
- **Registro trazable e inmutable de datos**  
- **Módulo de validación formal y pruebas reproducibles**

---

## Contexto
El Perú enfrenta una grave problemática de **pesca ilegal durante períodos de veda**, afectando la sostenibilidad marina y la economía nacional. Este trabajo demuestra cómo la **aplicación de métodos formales** puede fortalecer la confianza y exactitud en los sistemas de vigilancia pesquera, contribuyendo a la protección de los recursos marinos y al cumplimiento de las regulaciones nacionales e internacionales.
